# mget

`mget x y -> tile_id`

## Parameters

* **x, y** : map coordinates

## Returns

* **tile_id** : returns the tile id at the given coordinates

## Description

This function returns the tile at the specified [MAP](ram#map) coordinates, the top left cell of the map being (0, 0).
